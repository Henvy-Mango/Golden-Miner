# Golden-Miner
王者荣耀刷金币

## 使用说明
- 手机需开启USB调试模式，允许电脑调试，允许模拟点击。
- 电脑安装好手机驱动。
- 电脑上需要安装[Python3.6](https://www.python.org/downloads/)，并在安装界面勾选Add to PATH。
- 刷金币进入冒险模式，选择`魔女的回忆`，进入有闯关的按钮的界面，运行`run.bat`。
- 通关一次可获得19金币。
- 通关222次可获得一周4200上限金币

## 注意事项
- 请务必根据自己手机性能调整参数

## 调整方法 
- 使用文本编辑器打开kog.py
- 修改step_wait以调整等待时间

```python
# 各步骤等待间隔
#step_wait[0]为加载时间，不同手机加载速度不同
#step_wait[1]为战斗时间
#step_wait[2]为结算时间
#小米6的适配参数为[12,20,2]
step_wait = [12,20,2]
```
- 重复`加载 -> 战斗 -> 结算`流程
